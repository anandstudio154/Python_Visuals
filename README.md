# Interactive Particle Simulation

## Description
A visually engaging particle simulation where particles are attracted to the mouse cursor and repelled when too close.

## Features
- Real-time particle motion
- Mouse interaction (attraction + repulsion)
- Smooth trailing effects
- Built using Python and Pygame

## How it Works
- Particles move toward the mouse using attraction forces
- When particles get too close, they are repelled
- This creates dynamic and natural-looking motion

## How to Run
1. Install pygame:
   pip install pygame
2. Run:
   python main.py
